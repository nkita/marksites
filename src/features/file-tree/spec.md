# file-tree仕様

## 責務

ファイルツリー、常設ファイルサイドバー、パンくず、更新日時と、それらに必要なブラウザ動作を所有する。

## ファイル

- `index.ts`: マークアップ生成とブラウザスクリプトを構成する内部エントリポイント。

出力互換性を保ちながら分割する場合は、マークアップ、スタイル、ブラウザ動作をこのフォルダ内に置く。

ファイルは有効コメント件数をバッジ表示し、各ディレクトリは配下の全ファイルにある有効コメント件数の合計を再帰的にバッジ表示する。0件のバッジは表示しない。閉じたディレクトリでは配下の合計バッジを表示し、開いたディレクトリでは子要素のバッジとの重複を避けるため合計バッジを非表示にする。

ファイルパスのコピーはセキュアコンテキストでClipboard APIを優先し、権限拒否を含む書き込み失敗時は一時textareaと`execCommand("copy")`へフォールバックする。両方が失敗した場合だけ失敗として表示する。
